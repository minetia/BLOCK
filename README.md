지금까지 우리가 함께 만든 [THE BLOCK] 시리즈의 전체 시놉시스와 게임 소개입니다.
All.html을 실행했을 때, 플레이어가 경험하게 될 거대한 서사시를 요약했습니다.
​🎬 프로젝트 명: THE BLOCK (더 블록)
​"괴물을 잡으려다 괴물이 된 남자의 이야기"
​1. 장르 및 형식
​장르: 사이버펑크 누아르 / 인터랙티브 무비 / 웹 시뮬레이션
​형식: 별도 설치 없이 웹 브라우저에서 실행되는 옴니버스 HTML 게임
​플레이 타임: 약 10~15분 (엔딩 포함)
​2. 전체 줄거리 (Storyline)
​몰락한 천재 트레이더 **'강한준'**이 의문의 비트코인 생존 게임에 초대받으며 시작됩니다. 그는 가상의 낙원(메타버스)이 거짓임을 깨닫고, 황무지(현실)를 거쳐 시스템의 설계자 **'오상무'**에게 도전합니다. 결국 승리하지만, **권력의 맛을 본 주인공이 새로운 악(God)으로 타락하며 끝없는 순환(Loop)**을 암시합니다.
​3. 에피소드별 핵심 내용
​EP 01: SATOSHI CITY (시작)
​비트코인 등락에 목숨을 건 롱/숏 홀짝 도박. 실패 시 즉시 청산(사망).
​EP 02: THE REVENGE (해킹)
​시스템의 오류(Glitch)가 되어, 차트를 강제로 멈추고 방화벽을 뚫는 해킹 액션.
​EP 03: METAVERSE (심리)
​돈을 절대 잃지 않는 공포스러운 가짜 천국. 도망가는 '로그아웃' 버튼을 잡아 탈출해야 함.
​EP 04: REALITY (생존)
​모든 것이 파괴된 사막. 배터리를 관리하며 비밀번호를 해독하고 드론을 피하는 텍스트 어드벤처.
​EP 05: ZERO SUM (보스전)
​최종 보스 오상무와의 1:1 대결. 서로의 HP(자산)를 걸고 벌이는 50:50 확률 게임.
​EP 06: NEW GOD (결말)
​오상무를 제거하고 새로운 신이 된 당신. 시장을 폭등/폭락 시키며 개미들을 조종하다, 과거의 자신과 똑같은 새로운 도전자를 마주함.
​4. 관전 포인트
​몰입감 넘치는 UI 변화: 에피소드마다 차트, 터미널, 파스텔톤, 레트로 등 전혀 다른 디자인으로 전환됩니다.
​철학적 메시지: "승자가 모든 것을 독식하는 제로섬 게임의 비극"을 담고 있습니다.
​이 설명은 친구들에게 게임을 보여줄 때나, 포트폴리오로 정리할 때 사용하시면 됩니다! 🕹️
